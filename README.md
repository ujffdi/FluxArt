Standard React web project template built with Vite, React, and TypeScript.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Project Layout

```text
src/
  App.tsx
  App.css
  index.css
  main.tsx
  assets/
```

Use `npm run dev` for local development and `npm run build` to type-check and create the production bundle in `dist/`.
