# Separate asset history retention from physical deletion

FluxArt V1 will separate visible asset history from physical object deletion. Free Users keep visible history for generated and user uploaded assets for 7 days or 20 assets, whichever is stricter, so the asset center does not become unlimited storage. Credit Pack Users and Pro Members keep paid-generated assets long-lived in V1. User deletion sets deletedAt first, while MinIO object cleanup can run asynchronously later. Deleting a source asset does not cascade to assets generated from it; those derived assets keep their history while the UI can show that the source was deleted. Billing, ledger, order, and image task records remain available for audit.
