# Add Pro membership entitlements and downloads

## What to build

Sell and fulfill Pro membership cycles, grant monthly Promotional Credits, apply Pro download benefits, and record commercial authorization snapshots on eligible assets.

## Acceptance criteria

- [ ] The billing UI can start a Pro membership payment flow for the V1 CNY 69 monthly plan.
- [ ] Verified Pro payment notifications create or extend Membership Cycles.
- [ ] Each paid cycle grants 1000 monthly Promotional Credits that expire at cycle end.
- [ ] Pro Members receive 300 included HD no-watermark downloads per month.
- [ ] Beyond the Pro fair-use cap, HD no-watermark downloads cost 5 credits per image.
- [ ] Non-Pro HD no-watermark downloads cost 5 credits per image.
- [ ] Pro-generated assets store an Entitlement Snapshot and show the commercial authorization statement.
